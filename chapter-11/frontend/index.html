                Mailgun Setup Instructions
            

            

                
    
                    
  1. Go to mailgun.com and create a free account
    2. 
                    
  2. Get your API key from Settings > API Keys
    3. 
                    
  3. Get your sandbox domain from Sending > Domains
    4. 
                    
  4. Add authorized recipients in Sending > Overview > Authorized Recipients (sandbox only)
    5. 
                    
  5. Copy .env.example to .env and fill in your values
    6. 
                    
  6. Restart the server
    7. 
                

                

                    Note: Without Mailgun configured, emails are simulated and verification links are shown in the response.
