Token Flow

            

                
1. LOGIN → Get access_token (15 min) + refresh_token (7 days)
2. API CALL → Send access_token in Authorization header
3. ACCESS EXPIRED → Call /refresh with refresh_token → Get new access_token
4. LOGOUT → Blacklist refresh_token (can't be used anymore)
5. REFRESH EXPIRED → Must login again
