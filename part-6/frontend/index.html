How Frontend Gets Token from URL

In real apps, user clicks a link like:

http://yoursite.com/reset-password?token=abc123def456...

Frontend JavaScript extracts the token:

// Get all URL parameters
const params = new URLSearchParams(window.location.search);

// Get the 'token' parameter
const token = params.get('token');

// Now you have the token!
console.log(token);  // "abc123def456..."
