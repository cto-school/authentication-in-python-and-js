Mailgun Setup Instructions
  1. Go to mailgun.com and create free account
  2. Get your API Key from Dashboard > API Keys
  3. Get your sandbox domain from Sending > Domains
  4. Add your email as "Authorized Recipient" (for sandbox)
  5. Create .env file in backend folder: 
    MAILGUN_API_KEY=your-api-key
MAILGUN_DOMAIN=sandbox123.mailgun.org
MAILGUN_SENDER=Test <postmaster@sandbox123.mailgun.org>
FRONTEND_URL=http://localhost:5500
  6. Restart the server
